Você está prestes a aprender Seletores CSS! Seletores são como você escolhe a qual elemento aplicar estilos.

Anexo 1 - Uma Regra CSS

p { 
   margem inferior: 12px; 
}

Aqui, o "p" é o seletor (seleciona todos os elementos <p>) e aplica o estilo margin-bottom.

Para jogar, digite um seletor de CSS no editor abaixo para selecionar os itens corretos na mesa. Se você acertar, avançará para o próximo nível.

Passe o mouse sobre os itens na tabela para ver sua marcação HTML.

O atributo termina com o seletor

Selecione todos os elementos com um valor de atributo que termine com caracteres específicos

[atributo$="valor"]

Exemplos

img[src$=".jpg"] seleciona todas as imagens para exibir uma imagem .jpg .
